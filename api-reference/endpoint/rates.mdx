## ğŸ§  Why This Matters
When using the **offramp flow**, you must first call `approve()` for the token. Use this `/rates` route to calculate the amount of tokens (e.g. USDC) to approve:

```
approve_amount = amount_fiat / marked_up_rate
```

This ensures the user has approved **enough tokens** before calling `/orders/create` with `order_type: 1`.
